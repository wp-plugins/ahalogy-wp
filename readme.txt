Adds the Ahalogy Engagement code to your site with no fuss.

== Description ==

Designed to be very simple to use and reliable.

== Installation ==

1. Download the latest zip file and extract the directory.
2. Upload the this directory inside your `/wp-content/plugins/` directory.
3. Activate 'Ahalogy WP' on the 'Plugins' menu in WordPress.
4. Click 'Settings' underneath the plugin or navigate to Settings > Ahalogy and insert your Ahalogy User ID (0000000000-yourid) and click the checkbox to turn it on.
5. That's it.
